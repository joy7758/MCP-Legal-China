# MCP-Legal-China ⚖️

> **[EN]** The first China-specific Legal Plugin for Claude Cowork (MCP-based).
> **[CN]** 首个适配 Claude Cowork 的中国法律增强插件（基于 MCP 协议）。

---

## 🌟 为什么需要本项目？
Anthropic 官方发布的 Legal Plugin 主要基于英美法系。本项目旨在填补中国法环境下的空白，让 Claude 真正理解中国法律逻辑。

- **深度适配：** 基于 2026 年最新司法解释及《民法典》。
- **实时合规：** 通过 MCP 协议对接天眼查/企查查 API 进行主体背景调查。
- **本地化隐私：** 合规规则库支持本地部署，确保合同机密不外泄。

## 🗺️ 路线图 (Roadmap)
- [ ] **Phase 1:** 核心 MCP 协议框架搭建 (In Progress)
- [ ] **Phase 2:** 中国《民法典》常用合同模板库导入
- [ ] **Phase 3:** 自动化风险标注引擎 (支持中文 PDF/Word)
- [ ] **Phase 4:** 对接本地法律数据库 (裁判文书网/法规库)
