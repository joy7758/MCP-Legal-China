## 🏗️ 技术架构

### 核心技术栈
- **语言**: Python 3.10+
- **协议**: MCP (Model Context Protocol)
- **框架**: FastMCP (简化开发)
- **数据**: JSON/Markdown (规则存储)

### 依赖包
```
mcp>=1.0.0           # MCP SDK
fastmcp>=0.1.0       # FastMCP 框架
requests>=2.31.0     # HTTP 请求
python-dotenv>=1.0.0 # 环境变量
pandas>=2.0.0        # 数据处理
```

### MCP Server 架构
```python
MCP Server
├── Tools (工具)
│   ├── check_contract_risk()      # 合同风险检查
│   ├── get_company_risk_report()  # 企业风险报告
│   └── search_legal_cases()       # 法律案例搜索
│
├── Resources (资源)
│   ├── civil_code/                # 民法典
│   ├── contract_templates/        # 合同模板
│   └── legal_precedents/          # 法律先例
│
└── Prompts (提示词)
    ├── contract_review_flow       # 合同审查流程
    └── risk_assessment_template   # 风险评估模板
```
